เมื่อเปิดใช้งานจะมีตัวโปรแกรมแยกมาเป็น 2 ส่วนโดยส่วนแรกคือ
command prompt และส่วนที่2คือ ตัว Browser Google chrome
โดยการทำงานจะอยู่ใน command prompt จะแสดงหมวดหมู่สินค้าทั้งหมด
มีเลขกำกับหมวดหมู่อยู่หน้าชื่อหมวดหมู่การใช้งานให้เราใส่ตัวเลขของหมวดหมู่ที่
ต้องการดึงข้อมูลลงไปแล้วกด enter จากนั้นรอตัว brwoser ทำงานจนเสร็จ
แล้วตัว browser จะปิดลงไปเองเมื่อจบการทำงาน ซึ่งไฟล์ที่ได้จะอยู่ใน folder
เดียวกับไฟล์ .exe **ต้องมี browser google chrome ในการใช้งานโปรแกรมนี้
